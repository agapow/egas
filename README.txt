About the EgAS: the Epigenetic Association webservice
-----------------------------------------------------

EgAS is a webtool for serving experimenetally derived associations between
genomic variants (SNPs) and methylation sites. It allows searching, uploading,
downloading and editting.

EgAS was created by Paul Agapow (data Science Institute, Imperial College)
and Benni Lehne (Dept. Medicine, Imperial College). It is based on the
Flask and Flask Appbuilder frameworks, and written in Python.

See accompanying files for licensing and installation information.
